# C Recursion - Power of a Number

## 📌 Description
This project calculates the power of a number using recursion.

- Recursive Function
- Base Condition
- Power Calculation
- Beginner Friendly

## 💻 Example

Input:
Base = 2
Exponent = 5

Output:
32
